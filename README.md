# Sorting-Band-Names-without-articles

## Description

An exercise from Wes Boz's JS 30 Challenge. Using JS to list band names in alphabetical order without using the articles to order the list.

| Technologies used:    |
| --------------------- |
| Html, Vanilla JS, Css |
